# Changelog

## 0.2.0 - 2026-06-30

- 将项目定位收敛为可访问性配色审计与 Design Token 编译库。
- 修正 WCAG 相对亮度算法，加入标准 sRGB 分段线性化。
- 对比度阈值改用未舍入比率判断，避免 4.499:1 被误判为通过。
- 新增普通/大号文本 AA、AAA 分级与批量语义角色审计。
- 新增不合格前景色的确定性修复建议。
- 新增语义主题、DTCG 2025.10 颜色对象及 CSS 变量输出。
- 新增社区项目差异说明、架构文档和端到端 CLI 演示。
- 测试增至 22 项。

## 0.1.0 - 2026-06-11

- 初始化 MoonColorKit 基础结构。
- 增加 RGB、HEX、HSL、颜色混合、渐变和调色板能力。
- 增加 CLI、CI、Issue/PR 模板和项目文档。
