# MoonColorKit

MoonColorKit 是一个面向 MoonBit 的颜色空间与可访问性调色基础库。

项目方向和前面的 MoonNavKit、MoonSketchKit、MoonLexKit、MoonChartKit、MoonSignalKit、MoonMatrixKit、MoonGeoKit 都不同。它聚焦 RGB、HEX、HSL、亮度、对比度、调色板和渐变生成，适合设计工具、可视化系统、主题生成器、WebAssembly UI 和教学算法复用。
