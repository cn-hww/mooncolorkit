# 初审意见整改说明

## 初审意见

需要说明 MoonColorKit 与 Mooncakes 社区项目 `bobzhang/colors@0.8.1` 的异同。

## 整改结论

项目已从宽泛的“颜色空间与调色基础库”收敛为“可访问性配色审计与 Design Token 编译库”。RGB/HSL 等能力保留为兼容基础层，不再作为申报主题和主要工作量。

## 已完成的实质修改

1. 修复原有亮度算法，按标准对 sRGB 通道进行分段线性化。
2. 使用未舍入比率判断 WCAG 3.0、4.5、7.0 阈值。
3. 新增普通文本/大号文本 AA、AAA 分类。
4. 新增语义角色批量审计、汇总统计与 JSON 报告。
5. 新增不合格前景色的确定性修复建议。
6. 新增语义主题自审计、DTCG 2025.10 JSON 和 CSS 变量输出。
7. 新增临界阈值、批量审计、修复器和 Token 格式测试。
8. 新增 `docs/RELATED_WORK.md`，逐项说明与社区项目的边界和互补关系。

## 可复核结果

- `moon fmt --check`：通过。
- `moon check`：通过。
- 默认、JS、Wasm、Wasm-GC：每组 22 项测试全部通过。
- CLI 示例：将白底上的 `#9A9A9A` 修复为 `#767676`，对比度由 2.81:1 提升至 4.54:1。
- DTCG 输出包含 `colorSpace`、`components`、`alpha` 和 `hex`。
