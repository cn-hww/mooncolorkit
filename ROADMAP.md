# MoonColorKit Roadmap

## 0.2.x 审计核心

- 完成 WCAG 2.x 文本对比度、批量审计和修复建议。
- 完成语义主题、DTCG 2025.10 JSON 与 CSS 输出。
- 建立跨 JS/Wasm/Wasm-GC 的确定性测试。

## 0.3.x 设计系统门禁

- 增加 UI 组件和图形对象的 3:1 非文本对比度规则。
- 增加主题差异报告、审计策略配置和 CLI 退出码。
- 增加 Design Token 别名解析与循环引用诊断。

## 0.4.x 感知颜色

- 增加 OKLab/OKLCH 表示和色域映射。
- 增加 Delta E 2000 与调色板可区分性检测。
- 增加常见色觉缺陷模拟与图表序列配色检查。

## 长期方向

- 保持核心包后端中立和零平台 IO。
- 建立公开基准数据集与跨实现差异测试。
- 提供与社区颜色类型、UI 框架和构建工具的薄适配包。
